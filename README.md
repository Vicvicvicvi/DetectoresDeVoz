# DetectoresDeVoz
Detectores de voz
# ...
Este código utiliza la biblioteca SpeechRecognition para capturar el audio del micrófono y luego intenta reconocer el texto hablado utilizando el servicio de reconocimiento de voz de Google. Si el reconocimiento es exitoso, muestra el texto reconocido en la consola. Si ocurre algún error, muestra un mensaje de error.
